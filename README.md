# human_face_emotion_identify
数据集采用kaggle上面的公共数据集FER-2013，基线模型为自己搭建一个简单的卷积神经网络（CNN），采用迁移学习，微调预训练的Resnet18，实现
(1)单个人物图片情绪识别
(2)人物视频情绪识别
(3)人物情绪脸部热力图
(4)多个人物图片情绪识别
(5)可视化界面
# 基于深度学习的猫狗情绪识别系统
💻 计算机视觉 | 机器学习 | 大作业项目

## 📝 项目介绍
本项目是一个基于卷积神经网络（CNN）的猫狗情绪识别系统。
通过对猫狗脸部图像进行训练，实现自动识别宠物的情绪状态，包括：
开心、生气、放松、悲伤等常见情绪。

系统可用于宠物行为分析、智能宠物设备、人机交互等场景。

## ✨ 功能特点
- 支持识别猫狗脸部情绪
- 实时图片识别 / 批量识别
- 可视化界面展示结果
- 高精度深度学习模型
- 简单易用，一键启动

## 🛠️ 技术栈
- 开发语言：Python
- 深度学习框架：TensorFlow / PyTorch
- 图像处理：OpenCV
- 数据集：Kaggle 猫狗情绪数据集
- 模型：CNN / ResNet / EfficientNet
